EasyDay
=======

A simple and light day picker component for React.

## Installation

npm install easy-day --save

## Usage - RequireJS

var EasyDay = require('easy-day');
React.render(EasyDay);

## Usage - ES6

import EasyDay from 'easy-day'
React.render(EasyDay);

## Tests

npm test
